Les bases de HTML

L'HTML c'est quoi ?

L'HTML c'est quoi ?

L’Hypertext Markup Language, généralement abrégé HTML, est le format de données conçu pour représenter les pages web (on est sur du contenu).

C’est un langage de balisage permettant d’écrire de l’hypertexte, d’où son nom. HTML permet également de structurer sémantiquement et de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des formulaires de saisie, et des programmes informatiques.

L'HTML fait quoi ?

Il permet de créer des documents interopérables avec des équipements très variés de manière conforme aux exigences de l’accessibilité du web.

L'HTML avec quoi ?

Il est souvent utilisé conjointement avec des langages de programmation (JavaScript) et des formats de présentation (feuilles de style en cascade). HTML est initialement dérivé du Standard Generalized Markup Language (SGML).

Les normes HTML

  • HTML VS xHTML
  • Les variantes Transitionnal VS Strict (pas d'éléments dépréciés)
  • Le consortium derrière l'HTML : le W3C
  • L'accessibilité :
    Elle met en jeu 3 acteurs : le graphiste, l'intégrateur et le rédacteur (souvent le client).

HTML, un langage à 5 versions

HTML 1 : première version créée par Tim Berners-Lee en 1991.

HTML, un langage à 5 versions

HTML 2 : deuxième version du HTML (1994-1996). Cette version établit les bases des versions suivantes. Les règles et le fonctionnement de cette version sont donnés par le W3C.

HTML, un langage à 5 versions

HTML 3 : apparue en 1996, cette nouvelle version rajoute de nombreuses possibilités au langage comme les tableaux, les applets, les scripts, le positionnement du texte autour des images, etc.

HTML, un langage à 5 versions

HTML 4 : version la plus répandue. Elle apparaît en 1998 et propose l'utilisation de frames, des tableaux plus complexes, des améliorations sur les formulaires, etc. Mais surtout, cette version permet pour la première fois d'exploiter des feuilles de style.

HTML, un langage à 5 versions

HTML 5 : LA dernière version. Assez peu répandue cependant elle apporte de nombreuses améliorations comme la possibilité d'inclure facilement des vidéos, un meilleur agencement du contenu, de nouvelles fonctionnalités pour les formulaires, etc.

La mise en page

HTML et CSS réunis

Modèle de boite

Est appelé boite l'espace occupé par un éléments HTML de type boite (div, section, etc..) par opposition avec un éléments de type inline (a, span, etc..) qui s'insert dans le flux de texte.

Les balises structurantes de HTML5

  • <header> : l'en-tête
  • <footer> : le pied de page
  • <nav> : principaux liens de navigation
  • <section> : une section de page
  • <aside> : informations complémentaires
  • <article> : un article indépendant

Les fondamentaux de HTML5

HTML, et maintenant ?

Le futur de l'HTML

HTML5 est la prochaine révision majeure d'HTML. Cette version est encore en développement au jour d'aujourd'hui.

Le langage comprend également une couche applicative avec de nombreuses API ainsi qu'un algorithme afin de pouvoir traiter les documents à la syntaxe non conforme.

L'histoire d'HTML5

Le travail a été repris par le W3C en mars 2007 avec des étapes en 2011 pour la clôture des ajouts de fonctionnalités et courant 2014 pour la finalisation de la spécification. W3C encourage les développeurs Web à utiliser HTML5 dès maintenant.

CSS c'est quoi ?

Le CSS c'est quoi ?

CSS (Cascading Style Sheets : feuilles de style en cascade) est un langage informatique qui sert à décrire la présentation des documents HTML et XML.

Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.

Les bases du Javascript

Rendre les pages intéractives

Le JavaScript c'est quoi ?

Le javascript est un langage de programmation de scripts orienté objet.

Le langage a été créé en 1995 par Brendan Eich pour le compte de la Netscape Communications Corporation, qui s'est inspiré de nombreux langages, notamment de Java mais en simplifiant la syntaxe pour les débutants

Le JavaScript avec quoi ?

Le javascript est un langage interpreté par votre navigateur Web. Il est donc naturellement utilisé majoritairement sur internet pour dynamiser des pages web HTML.

Il existe une utilisation nouvelle côté serveur avec NodeJS par exemple.

Le JavaScript pourquoi ?

  • Ajouter de l'interactivité pour l'utilisateur
  • Dynamiser le contenu
  • Fluidifier la navigation
  • Améliorer les performances

Introduction au DOM

Le Document Object Model (ou DOM) est un standard du W3C permet à des programmes informatiques et à des scripts d'accéder ou de mettre à jour le contenu, la structure ou le style de documents XML et HTML1. Le document peut ensuite être traité et les résultats de ces traitements peuvent être réincorporés dans le document tel qu'il sera présenté.

Javascript & HTML5

Quelle frontière sépare les deux langages ?

Une frontière mince

Pour certains fonctionnalités natives d'HTML5, l'utilisation de JavaScript sert uniquement de mise à niveau des anciens navigateurs.

Une frontière mince : les formulaires

La page HTML

Créez un formulaire dans votre page, celui ci doit être composé de

  • Un champ texte qui doit respecter un pattern précis (pattern="[A-F][0-9]{5}")
  • Un champ date requis
  • Un champ URL avec un placeholder
  • Un bouton de soumission de formulaire

Les API HTML5

Comme nous l'avons évoqué précédemment, HTML5 intègre une couche applicative avec la mise à disposition d'API.

Celles ci rendent la frontière entre HTML5 et Javascript moins mince puisque le JS permet d'exploiter les API mis à disposition par HTML5

La géolocalisation

Le navigateur peut demander à l'utilisateur s'il souhaite partager sa géolocalisation.

Cette information est ensuite accessible via Javascript et permet de profiter de cette donnée pour un affichage ou une gestion de contenu différente.

La géolocalisation

Utilisez l'API de géolocalisation. Ajouter une zone de texte (textarea) dans votre formulaire.

                            
function maPosition(position) {
    var infopos = "Position déterminée :\n";
    infopos += "Latitude : "+position.coords.latitude +"\n";
    infopos += "Longitude: "+position.coords.longitude+"\n";
    infopos += "Altitude : "+position.coords.altitude +"\n";
    $('#geolocalisation').html(infopos);
    /* Inserer ici le contenu de l'adresse dans votre zone de texte */
}

if(navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(maPosition);
}
